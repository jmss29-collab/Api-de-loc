# Api-de-loc
Esta API tem como objetivo cadastrar e consultar informações de localização, incluindo:Latitude, Longitude, Cidade, Estado e País.A API foi desenvolvida em Django + Django REST Framework, com documentação automática em Swagger e Redoc. Ela permite criar, listar, atualizar e excluir registros, garantindo um modelo simples e funcional para estudo e prática de desenvolvimento backend.
